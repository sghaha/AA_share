# 단계별 상세 가이드 — AI 카페 모바일 오더 시스템 (코스웍)

이 폴더는 [../과제_진행_가이드.md](../과제_진행_가이드.md)의 **Phase 0~5를 단계별로 쪼개서**, 각 단계에서 *무엇을 · 어떻게 · 무엇을 참고해* 하는지 세세하게 적어둔 실행 문서 모음입니다.

## 이 폴더를 쓰는 법
- 위에서부터 순서대로 진행합니다. 각 파일 맨 아래 **← 이전 / 다음 →** 링크로 이동하세요.
- 각 단계 파일은 다음 구조로 되어 있습니다:
  1. **이 단계가 의미하는 것** — 왜 하는지, 큰 그림에서의 위치
  2. **목표(Output)** — 이 단계가 만들어내는 산출물
  3. **인증과제 설계서 어디에 들어가나** — 최종 보고서 매핑
  4. **담당(3인)** — 우리 조 역할 분담
  5. **세부 진행 아이템** — 실제로 하는 일 (스텝바이스텝)
  6. **참고자료** — 클릭 이동
  7. **작성 팁 · 자주 하는 실수**
  8. **완료 체크리스트**
